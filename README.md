## Getting Started

### Prerequisites
- Node v18.18.0 or higher
- Rust v1.77.2 or higher
- Anchor CLI 0.30.1 or higher
- Solana CLI 1.18.17 or higher

### Installation
#### Clone the repo

```shell 
git clone https://github.com/EasyTraDefi/TraDefi.git 
cd TraDefi
```


#### Install Dependencies

```shell
 npm install
```

#### Start the web app
```shell
 npm run dev
```
